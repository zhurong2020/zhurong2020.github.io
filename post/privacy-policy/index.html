隐私政策

最后更新: 2026-01-11

欢迎访问有心言者博客（以下简称"本站"）。本隐私政策说明了我们如何收集、使用和保护您在访问本站时的个人信息。

1. 信息收集

1.1 自动收集的信息

当您访问本站时，我们可能通过以下方式自动收集某些信息：

  • Google Analytics：我们使用 Google Analytics 来分析网站流量和用户行为。收集的信息包括：
    • 您的 IP 地址（匿名化处理）
    • 浏览器类型和版本
    • 操作系统
    • 访问时间和页面浏览记录
    • 引荐来源
  • Google AdSense：我们使用 Google AdSense 展示广告。Google 可能会使用 Cookie 来提供基于您兴趣的广告。

1.2 您主动提供的信息

如果您通过评论系统（Gitalk）留言，我们可能收集您的 GitHub 用户名和评论内容。

2. Cookie 使用

本站使用 Cookie 和类似技术来：

  • 改善您的浏览体验
  • 分析网站流量（Google Analytics）
  • 提供个性化广告（Google AdSense）

您可以通过浏览器设置来控制或删除 Cookie。但请注意，禁用 Cookie 可能会影响网站的某些功能。

3. 信息使用

我们收集的信息用于：

  • 分析和改善网站内容和用户体验
  • 了解访客的兴趣和需求
  • 展示相关广告

4. 信息共享

我们不会出售、交易或以其他方式向外部方转让您的个人身份信息，但以下情况除外：

  • 与 Google Analytics 和 Google AdSense 共享匿名统计数据
  • 法律要求或为保护我们的权利

5. 第三方服务

本站使用以下第三方服务：

  • Google Analytics：Google 隐私政策
  • Google AdSense：Google 广告政策
  • Gitalk：基于 GitHub Issues 的评论系统，受 GitHub 隐私政策约束

6. 数据安全

我们采取合理的技术和组织措施来保护您的信息安全。本站通过 HTTPS 加密传输数据。

7. 您的权利

您有权：

  • 通过浏览器设置禁用 Cookie
  • 使用 Google 的广告设置来管理广告偏好
  • 联系我们删除您的评论

8. 儿童隐私

本站不面向 13 岁以下的儿童。我们不会故意收集儿童的个人信息。

9. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并注明最后更新日期。

10. 联系方式

如果您对本隐私政策有任何疑问，请通过联系页面与我们联系。
