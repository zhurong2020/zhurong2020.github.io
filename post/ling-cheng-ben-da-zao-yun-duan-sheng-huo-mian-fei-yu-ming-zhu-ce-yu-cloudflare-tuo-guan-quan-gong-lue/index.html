引言

独立域名不仅是个人网站、博客、品牌建设的重要工具，更是数字化时代的身份象征。通过本文，您将学习如何：

  1. 在us.kg上注册免费的独立域名。

  2. 使用GitHub账号完成KYC验证。

  3. 将域名托管至Cloudflare，提升网站性能和安全性。

拥有域名后，您可以：

  1. 使用域名访问个人主页，为您的内容提供更专业的呈现。

  2. 将系列文章介绍的开源工具（如私人云盘、家庭影音系统、在线笔记系统等）通过反向代理绑定至域名，实现便捷访问。

  3. 结合Cloudflare的CDN服务，加速网站加载速度并提升安全性。

  4. 配置独立子域名为不同服务提供专属入口，例如 cloud.example.com 访问私人云盘，media.example.com 访问影音系统。

无论是零基础用户，还是技术爱好者，都可以通过本教程轻松实现零成本搭建云端生活。

关于免费-我的观点是：

在选择免费服务时，需谨慎考虑其潜在的隐性成本。正如经济学家所指出的，免费的东西往往质量不高，可能存在隐私泄露、服务不稳定等问题。因此，如果您计划长期使用某项服务，建议在免费与付费之间找到平衡，选择最适合自己的方案。然而，对于短期尝试或体验新服务，免费或试用版本仍是不错的选择。关键在于明确自身需求，做出理性的选择。

前期准备

在开始之前，请确保您已完成以下准备工作：

  1. GitHub 账号：如果您还没有，请前往 GitHub 注册，详细注册教程可以参考Github使用指南：打开云端写作新世界。

  2. Cloudflare 账号：前往 Cloudflare 注册一个免费账号。

  3. 个人真实信息：在 us.kg 注册时，请使用您的真实姓名、地址和联系方式，以确保账号的合法性和安全性。

  4. 域名名称：提前想好您希望注册的域名，可以多想几个备用选项，以备域名已被注册的情况。

操作步骤

1. 注册 us.kg 账号

  1. 访问 us.kg 官方网站：打开 register.us.kg，点击页面底部的“Sign Up”进行注册。

  2. 填写注册信息：输入您的法定全名、邮箱地址、电话号码和详细地址。请确保所填信息的真实性。

  3. 验证邮箱：提交注册信息后，登录您的邮箱，找到来自 us.kg 的验证邮件，点击其中的验证链接以激活账号。

2. 通过 GitHub 进行 KYC 验证

  1. 登录 us.kg 账号：在 register.us.kg 输入注册时的邮箱和密码，点击“Next”登录。

  2. 选择 KYC 验证方式：首次登录需要完成 KYC 验证，选择三种方法中的第二种“GitHub KYC Verification”，再点击下方的“Proceed to Github”。

  3. 提交 GitHub Issue：按照屏幕的指引，会自动跳转到Github并新增一个“issue”。请在描述栏（“description”）的下方，填写您的 us.kg 用户名和申请域名的理由（如个人博客、项目展示等），然后点击“Submit new issue”来完成提交 Issue。

  4. 完成验证：提交后，勾选相关选项，并为项目点亮 Star。等待几分钟，审核通过后，您将在 Issue 中收到批准通知。


    如果一下子没有找到刚才的issue，可以点击上方的“Issues”，并在“Filters”中选择“Your issues”，然后选择“Closed”，再点击标题就能看到批准通知：


3. 申请并注册免费域名

通过验证后，重新进入 us.kg 的控制面板，即可申请并注册免费域名。

  1. 检查域名可用性：在 us.kg 控制面板中，选择“Domain Name Registration”，输入您想要的域名，点击“Check”检查其可用性，如果不可用，会出现以下画面：

    如果域名可用，会出现以下画面：

  2. 获取 Cloudflare Name Server 信息：打开 Cloudflare，登录后选择“添加新网站”，再输入刚才在 us.kg 上申请的域名，选择最下方的免费计划，点击继续“Continue”，再点击“Continue to activation”。



Cloudflare 将提供两条 Name Server 信息，请分别复制备用。

  1. 注册域名：返回 us.kg 的域名注册页面，在 Name Server 1 和 Name Server 2 中分别粘贴从 Cloudflare 获取的 Name Server 信息，点击“Register”完成域名注册。

4. 将域名托管到 Cloudflare

  1. 完成 Cloudflare 设置：在 Cloudflare 中，继续设置您的域名，确保 DNS 记录正确配置。点击“继续”完成设置。

  2. 等待 DNS 配置生效：通常需要几分钟至数小时，您可以在 Cloudflare 中查看域名状态，显示“Active”即表示托管成功。

  3. 在us.kg中查看域名状态：在 us.kg 的控制面板中，查看您的域名状态，在“My Domains”中会显示已经成功的域名。

    以后，可以通过us.kg网站主页上方的“Get US KG”来进入域名管理页面，查看域名的状态和续期情况。

使用域名搭建个人网站

请注意，拥有域名并不意味着可以直接通过该域名访问您的网站。您需要将域名与实际的网站内容关联。以下是两种常见的方法：

方法一：使用 GitHub Pages 搭建静态网站

  • 使用 GitHub Pages 搭建静态网站：如果您计划创建一个静态网站，可以使用 GitHub Pages 服务。将您的网站代码上传至 GitHub 仓库，并在设置中启用 GitHub Pages 功能，即可将您的域名指向该静态网站。详细教程请参考我们的系列文章Github Pages教程：开启你的博客新旅程。

方法二：在自己的 VPS 上使用 WordPress 搭建动态网站

  • 在 VPS 上使用 WordPress 搭建网站：如果您希望搭建一个动态网站，如博客或企业站点，可以在 VPS（虚拟专用服务器）上安装 WordPress。完成安装和配置后，将您的域名解析至 VPS 的 IP 地址，即可通过域名访问您的 WordPress 网站。购买 VPS 教程可以参考云端启航：轻松拥有自己的VPS开启你的云端之旅，其它教程敬请期待。

结论

通过以上步骤，您已成功在 us.kg 注册了免费的域名，完成了 GitHub KYC 验证，并将域名托管至 Cloudflare。这不仅为您的云端生活增添了一份便利，也为未来的数字化探索奠定了基础。请记得每年提前 180 天登录 us.kg 账号进行域名续期，以保持域名的有效性。

常见问题解答（FAQ）

  • us.kg 的域名真的永久免费吗？

    • 是的，每个账号可以注册 3 个域名，需每年提前 180 天手动续期一次，续期也是免费的。

  • 为什么需要使用 GitHub 进行 KYC 验证？

    • us.kg 采用 GitHub 进行 KYC 验证，以确保用户的真实性和域名的合理使用。

  • 域名托管到 Cloudflare 有哪些优势？

    • Cloudflare 提供免费的 CDN 服务，提高网站加载速度，并提供安全防护功能，提升网站的稳定性和安全性。
