# zhurong2020.github.io

有心 (YouXin) 博客的 GitHub Pages 静态站点存档。

**在线地址**: https://zhurong2020.github.io

## 关于

这是有心 (原有心言者/Zhurong) 博客的历史文章存档，包含 86 篇从 Gridea 静态博客生成器创建的文章。

**主站已迁移到**: https://www.arong.eu.org (WordPress)

## 内容分类

- 认知升级
- 技术赋能
- 全球视野
- 投资理财
- 云技术入门
- 开源应用

## 技术说明

- **生成工具**: Gridea 静态博客客户端
- **部署**: GitHub Pages
- **SEO**: 已配置 sitemap.xml, robots.txt
- **广告**: AdSense 集成 (审核中)

## 目录结构
